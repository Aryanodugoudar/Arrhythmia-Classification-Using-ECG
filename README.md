Predict different arrhythmia on ECG:
-N : Non-ectopic beats (Normal Beat)
-L : Left Bundle Branch Block
-R : Right Bundle Branch Block  
-A : Atrial Premature Contraction  
-V : Premature Ventricular Contraction
